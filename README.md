# kubevirtbmc.io

Landing page for the KubeVirtBMC project

## Development

Prerequisites: [Hugo Extended](https://gohugo.io/installation/), Node.js (for markdownlint), curl.

| Command | Description |
| ------- | ----------- |
| `make serve` | Start local dev server |
| `make build` | Build the site |
| `make lint` | Lint Markdown files |
| `make test` | Build and validate HTML |
| `make ci` | Run lint + test |
| `make clean` | Remove build artifacts |
